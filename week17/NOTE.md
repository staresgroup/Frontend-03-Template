# 工具链

- YEOMAN（脚手架生成器）

  官方网站：https://yeoman.io/

  主要概念：

  - User interactions
    - Prompts
  - file system
  - managing dependencies

- webpack

本质上，*webpack* 是一个现代 JavaScript 应用程序的*静态模块打包器(module bundler)*。当 webpack 处理应用程序时，它会递归地构建一个*依赖关系图(dependency graph)*，其中包含应用程序需要的每个模块，然后将所有这些模块打包成一个或多个 *bundle*。

具有如下四个主要概念：

- 入口(entry)

- 输出(output)

- loader

- 插件(plugins)
